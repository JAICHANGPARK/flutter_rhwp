## Web Editor Module

`RhwpWebEditor` can load upstream `@rhwp/editor` from a configurable ESM URL.
The example defaults to `https://esm.sh/@rhwp/editor`, but that module is not
vendored into this package. Production apps that self-host or bundle
`@rhwp/editor` should include the notices required by that bundled npm package
and its dependency graph.
